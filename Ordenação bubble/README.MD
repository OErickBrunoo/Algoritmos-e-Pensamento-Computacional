#  Código de ordenação usando 2 tipos de ordenação.


Esse projeto usa a ordenção bubble sort, insertion sort para ordenar de forma crescente os números do RGM. Ele também marca o tempo de execução e os passos dados até a ordenação completa.


---

## Introdução

De menira simples o usuário colocará o RGM (8 digitos) no console e ele retornará os números ordenados em ordem crescente nos 2 tipos de ordenção.


---

## Funcionalidades


Ele faz uma ordenação ultilizando o algoritmo bubble sort, insertion sort.
Ele marca o tempo de execução.
Ele conta os passos dados pela ordenação até a ordenação completa.

---


## Bibliotecas usadas

* <stdio.h>
* <string.h>
* <time.h>

---

## Exemplos de Entrada

Digite o RGM para bubble sort: 45619662 \
Digite o RGM para insertion sort: 45619662

## saída

metodo,N,passos,tempo_ms
Bubble,1245666939,0.000002,

metodo,N,passos,tempo_ms
Insertion,1245666918,0.000001,

## Considerações

Levando em conta essa situação onde temos 8 caracteres, o bubble não parece ser o melhor para essa situação, o insertion conseguiu organizar em um tempo menor e com menos passos, chegando mesmo resultado.
